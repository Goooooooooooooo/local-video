# 本地视频管理

扫描指定文件夹，获取视频文件，自动从TMDB根据文件名匹配视频信息和海报（需要TMDB API KEY）

更加方便管理本地既存的视频文件（支持显示mkv视频文件时长，MP4暂不支持（功能还没写））
